# Planet Creator: Slider Effects Reference

## Quick Visual Effects Guide

### 1. Surface Composition & Rock/Soil Color
**Slider Range**: 0-100 (Surface Density)  
**Color Picker**: Rock/Soil Color

| Slider Value | Visual Effect |
|---|---|
| **0%** | All water (pure ocean blue) |
| **25%** | Mostly water with small islands |
| **50%** | Earth-like balance (50% land, 50% ocean) |
| **75%** | Mostly desert/land with small oceans |
| **100%** | Total desert (no water) |

**Pro Tip**: Combine with ocean color picker for distinct "Earth", "Mars", or "Alien" worlds

---

### 2. Ocean & Water
**Slider Range**: 0-100 (Ocean Coverage)  
**Color Picker**: Ocean Color

| Slider Value | Visual Effect |
|---|---|
| **0%** | No water variation (flat ocean) |
| **30%** | Light ocean with some depth variation |
| **50%** | Medium depth (darker blue tones) |
| **75%** | Very deep (dark navy appearance) |
| **100%** | Maximum depth (nearly black water) |

**Pro Tip**: Use dark blue (#0a1f4f) at 100% for deep alien oceans

## Testing Methodology

### Test Each Slider Individually

1. **Reset** to defaults first
2. Move **one slider** from 0% â†’ 100%
3. Watch how the planet changes
4. **Compare 0% vs 100%** to see full range
5. Test various **color combinations** for that category

### Test Slider Combinations

1. Start with "Earth" settings above
2. Gradually change one slider at a time
3. Note how the planet transforms
4. Save interesting configurations to local storage

### Visual Quality Checklist

- [ ] Continents look natural (not random blobs)
- [ ] Ocean appears deeper when coverage increased
- [ ] Ice caps appear at poles (not scattered)
- [ ] Atmosphere creates visible halo
- [ ] Clouds overlay realistically
- [ ] Volcanoes glow with lava color
- [ ] Rings are visually distinct from planet
- [ ] Roughness adds texture detail
- [ ] All colors update in real-time
- [ ] Planet rotates smoothly

## Performance Tips

### For Smooth Sliders
- Move sliders **slowly** for immediate feedback
- Move sliders **quickly** to batch multiple changes
- Wait **~50ms** between slider movements for texture regeneration

### For Best Visual Quality
- View planet at **medium zoom** (not too close)
- Use **non-pure colors** (not pure #FFFFFF for everything)
- Combine **3-5 active sliders** (not all 8 at max)
- Test on **different backgrounds** (dark helps see atmosphere)

- **Neptune**: #4169E1 (royal blue), #1E90FF (dodger blue)
- **Jupiter**: #DAA520 (goldenrod), #CD853F (peru)

### Exotic Atmospheres
- **Red**: #DC143C, #FF6347
- **Cyan**: #00FFFF, #20B2AA
- **Purple**: #9370DB, #DA70D6
- **Orange**: #FF8C00, #FFA500
- **Green**: #00FF00, #7FFF00

### Pro Tip
Avoid pure colors (#FF0000, #00FF00, #0000FF) as they look artificial. Use slightly muted versions for realism.

## Troubleshooting

### "My planet looks flat/boring"
- Try moving **surface density** to create land/ocean contrast
- Increase **roughness** to add texture
- Combine multiple sliders (not just colors)

### "The ice caps look weird"
- Make sure **surface density** > 0% (need land for contrast)
- Try **ice coverage** at 10-50% (extreme values may clip with land)

### "Rings aren't visible"
- Check that **ring visibility** > 0%
- Try **brighter ring color** (#FFD700 vs #555555)
- Rotate planet to see rings from different angle

### "Sliders lag/freeze"
- Browser might be regenerating texture
- Wait 50-100ms between slider movements
- Close other browser tabs to free memory
